## ✦ About me

I work with Arduino and ESP32 for embedded systems to collect and control real-world data.  
Creating innovative solutions in IoT.

I connect devices to scalable backends and create reliable APIs, with focus on clean architecture, code organization, and user experience.

I also design and develop games in ROBLOX, including gameplay design and textures.

### ▸ Technical interests
- System integration between hardware and web platforms  
- Game systems design and texture creation  

### ▸ Beyond code
- Designing game ideas and mechanics  
- Exploring hardware projects and electronics  
- Learning new tools by building actual projects
